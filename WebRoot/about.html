About This Server

Information about the C# Web Server implementation

Server Details

This is a custom HTTP web server built with C# and .NET, implementing TCP socket communication.

Features:

  • Serves static HTML, CSS, and JavaScript files
  • Handles HTTP GET requests
  • Multi-threaded request processing
  • Security features preventing directory traversal
  • Proper HTTP status codes (200, 404, 405, 403)
  • Request logging with timestamps

Technical Specifications:

  • Port: 8080
  • Web Root: WebRoot/
  • Supported Extensions: .html, .css, .js
  • Default Page: index.html
